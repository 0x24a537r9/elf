MEAN.JS Changelog

Version 0.1(19/01/2013)

Bug Fixes

Several bug fixes including performance and other issues with Express configuration.

Folder Restrucure

  • Server.js is now application.js.
  • App folder is now Server folder.
  • app/config/authorization.js was removed, middlewares moved to controllers.
  • tests folder moved to server for Mocha tests, AngularJS Karma tests moved to module folder.
  • AngularJS application views and JS folders were removed, each module now supports its own folder structure to ensure better project development. To learn more about this change please visit the AngularJS Modules section in the documentation.

Templates Changes

  • JADE was replaced with doT.
  • Server side authentication views moved to AngularJS views.
  • Error pages were consolidated.

Design Changes

  • Bootstrap 3.0.3 support.
  • Views redesigned to support reponsive behaviour.

AngularJS Module Changes

AngularJS files were moved to ensure a better module system. To learn more about this change, please visit the AngularJS Modules section in the documentation.
