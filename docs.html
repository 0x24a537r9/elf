Overview

Thank you for downloading the MEAN.JS boilerplate!

This simple documentation will cover the basics of developing your MEAN application.

Before you begin we recommend you read about the basic building blocks that assemble a MEAN.JS application:

MongoDB
Go through MongoDB Official Website and proceed to its Great Manual, which should help you understand NoSQL and MongoDB better.
Express
The best way to understand express is through its Official Website, particularly The Express Guide, you can also go through this Stackoveflow Thread for more resources.
AngularJS
Angular's Official Website is a great starting point. You can also use Thinkster Popular Guide, and the Egghead Videos.
Node.js
Start by going through Node.js Official Website and this Stackoveflow Thread, which should get you going with the Node.js platform in no time.
When you're done with those resources and feel you understand the basic prinicpals continue to other sections.

Enjoy & keep us updated,
The MEAN.JS Team.
