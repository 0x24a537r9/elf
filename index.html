What is MEAN.JS?

MEAN.JS is a full-stack JavaScript boilerplate that helps you build fast, robust and maintainble web applications using MongoDB, Express, AngularJS, and Node.js.

Why MEAN.JS?

Developing a modern web application can be fun and productive, MEAN.JS will help you getting started and avoid useless grunt work, common pitfalls, and the frustration of working with bad practices.

Getting started

After downloading the source code go over the official documentation of MongoDB, ExpressJS, AngularJS, and Node.js, then continue with the MEAN.JS Documentation.
